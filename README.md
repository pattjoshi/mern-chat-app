# How to own this

### Must have in Yor System (STAP 1)
- node
- mongo
- npm


### STAP-2
`git clone https://github.com/pattjoshi/mern-chat-app.git`

###  STAP 3 (public is front-end) local host 3000
- `cd public`
- `yarn`
- `yarn start`

### STAP 4:- (server is back-end) local host 5000
- `cd server`
- `yarn`
- `yarn start`
